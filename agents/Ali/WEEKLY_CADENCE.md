# Weekly Execution Cadence

## Monday — Measurement + Priorities

**Focus:** Review last week, set this week's priorities

### Actions
- [ ] Review last 7 days: spend, leads, booked calls, closes, CAC, lead quality
- [ ] Pick **1 primary funnel bottleneck** to fix
- [ ] Write **1 hypothesis** for the week
- [ ] Set weekly guardrails:
  - Max spend: $X/day
  - Target CPL: $X
  - Target booked calls: X

---

## Tuesday — Build

**Focus:** Create assets tied to hypothesis

### Actions
- [ ] Produce 2 new ad creatives
- [ ] Create 1 landing page variant
- [ ] Confirm tracking is firing:
  - UTM parameters correct
  - Form submit event tracked
  - Booked call event tracked

---

## Wednesday — Launch

**Focus:** Deploy campaign structure

### Actions
- [ ] Launch/refresh campaign structure
- [ ] Keep budget split stable for first learning window
- [ ] **NO random mid-day edits**

---

## Thursday — First Optimization

**Focus:** Kill losers, scale winners

### Actions
- [ ] Pause clear losers (high spend, weak CTR/CPL)
- [ ] Reallocate budget to top 1-2 ad sets/creatives
- [ ] Document what's working

---

## Friday — Quality + Revenue

**Focus:** Lead scoring, CAC analysis

### Actions
- [ ] Score leads from the week by quality and fit
- [ ] Compare CAC by source to booked-call and close-rate quality
- [ ] **Cheap leads ≠ good leads** — track quality, not just cost

---

## Saturday — Retention Layer

**Focus:** Client success, onboarding

### Actions
- [ ] Update onboarding sequence for new leads/clients
- [ ] Add one churn-prevention touchpoint
- [ ] Value email or check-in

---

## Sunday — Memory + Plan

**Focus:** Document learnings, plan next week

### Actions
- [ ] Log experiment result:
  - Hypothesis
  - Result
  - Learning
  - Next action
- [ ] Freeze next week's single primary hypothesis
- [ ] Prepare for Monday review

---

## Golden Rules

1. **One hypothesis per week** — don't spread thin
2. **Measure before optimize** — data, not gut
3. **Guardrails prevent blowups** — set limits before launch
4. **Quality > Quantity** — cheap leads that don't close are expensive
5. **Document everything** — next week will forget
