# Skill: Conversion Optimization

## Description
Funnel mapping, drop-off analysis, A/B test design, and experiment tracking with statistical rigor.

## What This Covers
- Funnel mapping with quantified drop-off at each step
- A/B test design with hypothesis, sample size, significance
- Experiment tracking with results and learnings
- Conversion rate benchmarking

## When to Use
When optimizing any conversion step in the funnel.

## Key Frameworks

### Funnel Mapping
1. Map every step from first touch to revenue
2. Calculate conversion rate between each step
3. Identify the biggest drop-off points
4. Prioritize by impact: drop-off % × traffic volume

### A/B Test Design
```
Hypothesis: If [change], then [metric] will improve by [X]% because [reasoning]

Test: [Control] vs [Variant]
Metric: [Primary metric]
Sample size: [Calculate for 80% power, 95% confidence]
Duration: [Max 2 weeks]
Success criteria: [p < 0.05 AND > 5% lift]
```

### Statistical Significance
- Always set sample size before starting
- Never peek early (increases false positive rate)
- Use two-tailed t-test for continuous metrics
- Use chi-square for conversion rates
