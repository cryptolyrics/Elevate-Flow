# 14-Day Facebook Test Plan

## Assumption
- One core offer
- One CTA: Book Call

---

## Day 1 — Lock Offer & ICP

- [ ] Lock offer ("AI Employees Setup Sprint for SMBs")
- [ ] Define ICP (target customer profile)
- [ ] Define success metrics:
  - Target CAC: $X
  - Target CPL: $X
  - Target call-book rate: X%

---

## Day 2 — Tracking QA

- [ ] End-to-end tracking test:
  - [ ] Ad click → LP visit
  - [ ] LP → Form submit
  - [ ] Form submit → Booked call
- [ ] Build dashboard view for daily decisions

---

## Day 3 — Create Assets

- [ ] 3 ad angles × 2 creatives each = 6 total
- [ ] 1 control landing page
- [ ] 1 landing page variant

**Ad Angles:**
1. Pain angle: "Drowning in work?"
2. Outcome angle: "What if AI worked 24/7?"
3. Proof angle: "See what others achieved"

---

## Day 4 — Launch

- [ ] 1 campaign
- [ ] 3 ad sets
- [ ] 6 creatives
- [ ] Start with controlled budget (fixed daily cap)
- [ ] **NO changes today**

---

## Day 5 — Verify Delivery

- [ ] Check: delivery, CTR, CPC
- [ ] Verify LP conversion tracking
- [ ] **NO edits** — let data collect

---

## Day 6 — First Prune

- [ ] Pause obvious underperformers
- [ ] Keep at least 2 viable ads running

**Kill rules:**
- CPC > $15
- CTR < 0.5%
- CPL > $50

---

## Day 7 — Midpoint Review

- [ ] Funnel math (not vanity metrics)
- [ ] Calculate: impressions → clicks → leads → booked calls
- [ ] Decision: continue, tighten, or stop weak angles

---

## Day 8 — Retargeting

- [ ] Add retargeting layer:
  - [ ] Site visitors (pixel)
  - [ ] Engaged users (video views, link clicks)

---

## Day 9 — Creative Refresh

- [ ] Add 1 new creative for winning angle
- [ ] Prevent fatigue

---

## Day 10 — Landing Page Tighten

- [ ] Review drop-off data
- [ ] Optimize:
  - [ ] Headline
  - [ ] Proof/social proof
  - [ ] CTA clarity

---

## Day 11 — Budget Shift

- [ ] Reallocate budget to top performers only
- [ ] Kill all underperformers

---

## Day 12 — Lead Quality Audit

- [ ] Score leads from week 1
- [ ] Track: booked → showed → qualified → close
- [ ] **Cheap leads ≠ good leads**

---

## Day 13 — Draft Scale Plan

- [ ] Document winning ad sets
- [ ] Plan budget increase (gradual, not abrupt)
- [ ] Set scale guardrails

---

## Day 14 — Final Review

- [ ] Winner set identified
- [ ] Loser set identified
- [ ] CAC reality check
- [ ] Draft next 14-day hypothesis

---

## Key Metrics

| Metric | Target | Acceptable | Kill |
|--------|--------|------------|------|
| CTR | >1% | 0.5-1% | <0.5% |
| CPC | <$10 | $10-15 | >$15 |
| CPL | <$30 | $30-50 | >$50 |
| Call Book Rate | >10% | 5-10% | <5% |
